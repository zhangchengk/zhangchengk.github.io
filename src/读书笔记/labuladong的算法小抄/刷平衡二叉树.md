## 110-平衡二叉树

![](https://gitee.com/zhangchengk/image/raw/master/LeetCode/110/1.png)

![](https://gitee.com/zhangchengk/image/raw/master/LeetCode/110/2.png)

```java
class Solution {
    public boolean isBalanced(TreeNode root) {
        // 后序
        if (traverse(root) < 0){
            return false;
        } else {
            return true;
        }
    }

    int traverse(TreeNode node){
        if (null == node){
            return 0;
        }
        int leftH = traverse(node.left);
        int rightH = traverse(node.right);
        if (leftH == -1 || rightH == -1 || Math.abs(rightH - leftH) > 1){
            // 不合格
            return -1;
        } else {
            // 当前节点为root的子树合格
            return Math.max(leftH, rightH) +1;
        }

    }
}
```

## 108-将有序数组转换为二叉搜索树

![](https://gitee.com/zhangchengk/image/raw/master/LeetCode/108/1.png)

--

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return buildTree(nums, 0, nums.length-1);
    }

    TreeNode buildTree(int[] nums, int start, int end){
        // 递归找中位数
        if (start > end){
            return null;
        }
        int median = (end - start)/2;
        TreeNode node = new TreeNode(nums[median]);
        node.left = buildTree(nums, start, median -1);
        node.right = buildTree(nums, median +1, end);
        return node; 
    }
}
```

## 109-有序链表转换二叉搜索树

![](https://gitee.com/zhangchengk/image/raw/master/LeetCode/109/1.png)

--

将109转成108解题。
```java
class Solution {
    List<Integer> valList = new ArrayList();
    public TreeNode sortedListToBST(ListNode head) {
        while (null != head){
            valList.add(head.val);
            head = head.next;
        }
        return buildTree(0, valList.size() - 1);
    }

    TreeNode buildTree(int start, int end){
        if (start > end){
            return null;
        }
        int mid = (end + start)/2;
        TreeNode node = new TreeNode(valList.get(mid));
        node.left = buildTree(start, mid - 1);
        node.right = buildTree(mid + 1, end);
        return node;
    }
}
```

中序遍历+中位数
```java
class Solution {
    ListNode globalHead;

    public TreeNode sortedListToBST(ListNode head) {
        globalHead = head;
        int length = getLength(head);
        return buildTree(0, length - 1);
    }

    public int getLength(ListNode head) {
        int ret = 0;
        while (head != null) {
            ++ret;
            head = head.next;
        }
        return ret;
    }

    public TreeNode buildTree(int left, int right) {
        if (left > right) {
            return null;
        }
        int mid = (left + right + 1) / 2;
        TreeNode root = new TreeNode();
        root.left = buildTree(left, mid - 1);
        root.val = globalHead.val;
        globalHead = globalHead.next;
        root.right = buildTree(mid + 1, right);
        return root;
    }
}
```
