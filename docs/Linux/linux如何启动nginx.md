第一次启动：

在nginx安装目录下的sbin目录下执行：./nginx -c /usr/local/nginx/nginx/conf/nginx.conf

其中/usr/local/nginx/nginx/conf/nginx.conf是你自己的nginx.conf路径。

重新平滑启动：

在nginx安装目录下的sbin目录下执行： ./nginx -s reload
